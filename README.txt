------------
Description
------------

This module give more freedom to manage the price components form the commerce_price module from the Drupal Commerce solution (D7 version).
Currently, it's only available in French.

Ce module permet de manipuler plus facilement les composants des prix du module commerce_price de la solution Drupal Commerce (en version D7).


------------
Requirements
------------

- Drupal 7



------------
Usage
------------

Installer et activer le module comme un module Drupal 7 ordinaire.

La page de configuration se situe à :
FR: administration > Boutique > Configuration > Paramètres de Commande > Composants Prix Total
ENG : Administer > Commerce > Configuration > Order > Composants Prix Total
machine menu name : 'admin/commerce/config/order/price_components_options'

Elle requiert les droits d'administer les commandes pour y accéder.
